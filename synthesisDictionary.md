## 🎯 トーン基準

全フレーズは以下の基準で統一されています:

- **準具体レベル**: シーン限定しすぎず、行動の型として書く
- **断定温度**: 占術的事実は断定、解釈や予測は適度な余白
- **ポポの語尾**: 「〜ンダ」「〜ンダよ」を適度に
- **バーナム効果**: 多くの人が薄々自覚しているが言語化されていないことを書く
- **肯定的な着地**: 欠点指摘は才能の裏返しとして提示

| 偏財_正財 | 流動と堅実──広げる人と守る人の組み合わせンダ。 | 理想的な補完関係──片方が動き、片方が支える役割分担ができるンダよ。 | 相手の慎重さを「遅い」と感じたり、動きを「軽い」と感じたりしないことンダ。 |

| neutral | その他 | 喜神の重なりが少ない配置ンダ。運気の波がそれぞれ独立して動く関係ンダよ。 |

## 11. 特殊星ペア(主要)

キー: `specialStarPair[<自分の特殊星>_<相手の特殊星>]` → `{ intro, synergy }`

**主要29パターン + default** を収録。全組み合わせではなく、よく現れる組み合わせを中心に。

**重要**: `getSpecialStarPair()` ヘルパー関数を使用すると、`A_B` がなければ `B_A` を自動検索し、それもなければ `default` を返します。

### 収録されている特殊星

- 天乙貴人(てんおつきじん) - 人に助けられる星
- 華蓋(かがい) - 芸術と孤独の星
- 将星(しょうせい) - リーダーの星
- 駅馬(えきば) - 移動と変化の星
- 文昌貴人(ぶんしょうきじん) - 学問と表現の星
- 羊刃(ようじん) - 強烈な個性の星
- 紅艶(こうえん) - 魅力と華の星
- 天徳貴人(てんとくきじん) - 徳の星
- 月徳貴人(げっとくきじん) - 月の徳の星
- 魁罡(かいごう) - 強烈な運勢の星

代表例:

| 自分_相手 | intro | synergy |
|----------|-------|---------|
| 華蓋_天乙貴人 | あなたの華蓋(孤独と芸術の星)と、相手の天乙貴人(人に助けられる星)ンダ。 | あなたの孤独が相手の糧になり、あなたの一人時間を相手が受け入れられる関係ンダよ。 |
| 将星_華蓋 | あなたの将星(リーダーの星)と、相手の華蓋(芸術性の星)ンダ。 | あなたが前に立ち、相手が深める──表と奥の役割分担が自然にできる関係ンダよ。 |
| 羊刃_天乙貴人 | あなたの羊刃(強烈な個性)と、相手の天乙貴人(人徳の星)ンダ。 | あなたの強さを、相手の柔らかさが受け止めて丸くしてくれる関係ンダよ。 |

(全29パターン + default は `synthesisDictionary.js` を参照)

### 複数特殊星の扱い

2人がそれぞれ複数の特殊星を持つ場合、**最も強い組み合わせ1ペアを選択**する必要があります。選択優先順位の推奨:

1. 両者に特別な相性がある組み合わせ(例: 華蓋×天乙貴人)
2. 一方が持つ強い星(魁罡、羊刃など)
3. 同名の星がある場合(天乙貴人×天乙貴人など)
4. 残りの組み合わせ

この選択ロジックは `buildSpecialStarSection()` 側で実装します。

## 12. 流年テンプレート

キー: `annualFortune[タイプ]` → `string`(変数埋込形式)

**変数**:
- `{year}` - 年(例: 2026)
- `{element}` - 五行(例: '火')
- `{target}` - 対象('あなた' / '相手' / '2人')
- `{startYear}`, `{endYear}` - 期間指定

| タイプ | 使用条件 | フレーズ |
|--------|---------|---------|
| kishin_year | 一方の喜神が巡る年 | `{year}年は{element}の年──{target}の喜神が巡る追い風の年ンダよ。{target}にとってエネルギーが満ちる時期ンダ。` |
| imushin_year | 一方の忌神が巡る年 | `{year}年は{element}の年──{target}にとっては少し消耗しやすい時期ンダ。無理な決断は避けて整える期間に充てるといいンダよ。` |
| void_period | 空亡期 | `{startYear}年〜{endYear}年は{target}の空亡期ンダ。運気が一時的に弱まるから、新しい決断より関係を見つめ直す時期に向いているンダよ。` |
| deep_bond_year | 両者の喜神が重なる年 | `{year}年は2人の喜神が重なる貴重な年ンダ。関係を深める決断や形にする選択が、この時期なら自然に噛み合うンダよ。` |
| tension_year | 両者とも運気が整わない年 | `{year}年は2人とも運気が整わない時期ンダ。衝突しやすいから、大きな判断は避けて穏やかに過ごすことンダよ。` |

### 使用例

```js
import { fillTemplate } from '@/data/synthesisDictionary';

const msg = fillTemplate(

  { year: 2026, element: '火', target: '相手' }

//     相手にとってエネルギーが満ちる時期ンダ。"
```

```js
import SYNTHESIS_DICT_JA, {
  fillTemplate,

  getSpecialStarPair
```

```js
function buildSynthesisSection(self, partner) {
  const D = SYNTHESIS_DICT_JA;

  const selfSection = [

      D.identityModifier[`${self.identity}_${selfAlignment}`]

  return [selfSection, /* ... */, fiveSection, tgSection].join('\n\n');
}
```
